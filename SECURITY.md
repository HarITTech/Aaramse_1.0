# Security Policy

## Supported Versions

We provide security updates for the following versions of AaramSe. Please ensure you are using a supported version to receive critical security patches and updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in AaramSe, please report it to us at **security@aaramse.com**.

### What to Include in the Report:
- A description of the issue.
- Steps to reproduce the vulnerability (if applicable).
- Potential impact of the vulnerability.
- Any potential fixes or suggestions (optional).

### What to Expect:
- **Initial Response**: We will acknowledge receipt of your report within 48 hours.
- **Update Frequency**: We will provide status updates on the vulnerability resolution process every 7 days.
- **Resolution**: If the vulnerability is accepted, we will issue a patch as soon as possible. If declined, we will provide a clear explanation of the decision.

We greatly appreciate your help in keeping AaramSe safe for all users!
